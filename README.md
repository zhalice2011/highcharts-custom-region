# highcharts-custom-region
highcharts 中国行政地图大区划分--https://www.hcharts.cn/mapdata

## Introduction

中国的各种区域图。

# Region
| 大区名称 | 包含省份地区 |
| ------------------|:-------:|
| 华北地区               | 北京，天津，河北，山西，内蒙古 |
| 华东地区               | 上海，江苏，浙江，山东，安徽 |
| 东北地区               | 辽宁，吉林，黑龙江 |
| 华中地区               | 湖北，湖南，河南，江西 |
| 华南地区               | 广东，广西，海南，福建 |
| 西南地区               | 四川，重庆，贵州，云南，西藏 |
| 西北地区               | 陕西，甘肃，新疆，青海，宁夏 |
| 港澳台地区              | 香港，澳门，台湾 |

## Use

```
$ git clone https://github.com/zhalice2011/highcharts-custom-region.git

$ cd highcharts-custom-region

$ live-server
```
